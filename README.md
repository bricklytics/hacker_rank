# Hacker Rank

This repository to group all challenges done from Hacker Rank site using kotlin, primarily using kotlin language.


### Challenges
you can find the codes of each challenges done in: 

    src/main/kotlin 

The input files can be found in:
    
    src/main/inputfiles
